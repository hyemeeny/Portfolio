## ⭐ 프로젝트 소개

VS Code 스타일로 제작된 포트폴리오 사이트입니다.<br/> Home, About, Project, Contact의 네 섹션으로 구성되어 있으며,<br/> Project는 탭 형태로 분류해 다양한 프로젝트를 한눈에 확인할 수 있습니다.<br/> 각 프로젝트는 모달을 통해 상세 내용을 확인할 수 있도록 구현했습니다.
